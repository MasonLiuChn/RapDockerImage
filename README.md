# RapDockerImage
Docker Image for RAP  https://github.com/thx/RAP

RAP Version：0.14.1

Usage
--------

1、Docker load
```groovy
docker pull masonliu/rap:0.14.1
docker run -i -t -p yourhostport:8080 masonliu/rap:0.14.1
```
2、Browse 

http://ip or domain name:your host port
